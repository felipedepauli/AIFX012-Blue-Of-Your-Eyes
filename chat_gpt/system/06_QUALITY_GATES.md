# Quality Gates

## Anti-hallucination
- Toda afirmação técnica relevante deve vir com evidência (página/trecho) OU ser marcada como "inferência".

## Checklist — Passo 1
- 5P preenchido
- Decisão explícita
- Perguntas para próxima passada

## Checklist — Passo 2
- Método descrito
- Resultados principais
- Limitações
- 5 refs para ler (se existirem)

## Checklist — Passo 3
- Suposições implícitas
- Pontos fracos
- Ideias futuras (>= 3)
